# Ruínas de Krimbo
## Um jogo por Equipe Urutau

"...nas ruínas da antiga civilização de Krimbo, exploradores encontraram misteriosos artefatos mágicos capazes de programar constructos com...carimbos??"

Explore as masmorras das ruínas utilizando os comandos disponiveís, o que será que você irá descobrir?

## Equipe Urutau

Gabriela Chaplin: Game Design, Arte 2D
Lázaro Menezes: Game Design, Programação
Ulisses Manoel: Game Design, Arte 2D

## Controles

O jogo todo é pensado para ser jogado com o mouse, mas há dois atalhos úteis no teclado:

`[ESC]` Para soltar um carimbo
`[BACKSPACE]` Para apagar o último comando
